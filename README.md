📑 Control de Aportes PRO
Un sistema web ligero y eficiente para la gestión de pagos mensuales, diseñado para organizaciones, clubes o grupos pequeños que necesitan llevar un control de cuotas de sus integrantes.

🚀 Características Principal
Gestión de Usuarios: Registro rápido con validación de DNI y nombres.

Control de Pagos Interactivos: Sistema de "Toggle" (clic para marcar/desmarcar) para meses de Abril a Diciembre.

Diseño Inteligente (Mobile First): La interfaz se transforma en tarjetas verticales para un uso cómodo desde celulares.

Buscador en Tiempo Real: Filtra instantáneamente por nombre o número de DNI.

Reportes PDF: Generación automática de reportes profesionales en formato horizontal.

Backend Robusto: API REST construida con Node.js y Express, utilizando archivos JSON para persistencia de datos local.

🛠️ Tecnologías Utilizadas
Frontend: HTML5, CSS3 (Custom Variables, Flexbox, Grid), JavaScript Vanilla.

Backend: Node.js, Express.js.

Librerías: jsPDF y AutoTable para reportes.

Base de Datos: JSON (File System).

📦 Instalación y Uso
Clonar el repositorio:

Bash
git clone https://github.com/tu-usuario/control-pagos-pro.git
cd control-pagos-pro
Instalar dependencias:

Bash
npm install
Iniciar el servidor:

Bash
node index.js
Abrir la aplicación:
Abre el archivo index.html en tu navegador favorito.
